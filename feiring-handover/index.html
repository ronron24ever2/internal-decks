Phase 1 · pilot
Free, fully specified

One Hive L180, three months, two phases on same machine. Cost split frozen (§ 2.2). Acceptance criteria framework set; thresholds = addendum within three weeks of start (§ 2.3). Either party can exit on two weeks' written notice (§ 2.1).

Signed 28 Apr · not started
Phase 2 · operations
Intent only

Remote operation on two of Feiring's own machines from Kristiansand. Commercial terms (pricing, install cost, runtime, duration, service boundary) negotiated in separate Term Sheet / MSA before start (§ 3).

No commercials yet
Phase 3 · on-site centre
Intent only

Operations centre at Feiring; one operator supervises 4–6 machines after Hive training. Scope and investment in separate agreement when maturity allows (§ 4).
